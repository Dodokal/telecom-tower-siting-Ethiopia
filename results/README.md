# Results directory

This directory holds the small CSV outputs and reference results that are tracked in Git. Larger artefacts (intermediate rasters, trained models, model checkpoint .rds files) are excluded via `.gitignore` and regenerated by running the pipeline.

## What's tracked

| File | Produced by | Description |
|---|---|---|
| `r1_moran_results.csv` | `R/10_reviewer_response.R` | Moran's I of LightGBM residuals at five distance bands; addresses reviewer comment #5 |
| `r1_variogram_fit.csv` | `R/10_reviewer_response.R` | Fitted exponential semivariogram: range = 7.45 km, partial sill = 0.134, nugget = 0 |
| `r2_brier_scores.csv` | `R/10_reviewer_response.R` | LightGBM out-of-fold Brier score (0.001529); addresses reviewer comment #6.1 |
| `r4_equity_weight_sensitivity.csv` | `R/10_reviewer_response.R` | Spearman ρ between priority and equity gap under three weighting schemes; addresses reviewer comment #8 |
| `r5_threshold_sensitivity.csv` | `R/10_reviewer_response.R` | Greenfield-priority area under a 3×3 grid of percentile × radius thresholds; addresses reviewer comment #19 |

## What's NOT tracked (excluded via .gitignore)

- `models/*.rds` — fitted base learners and stacked ensemble (~500 MB)
- `checkpoints/*.rds` — script checkpoint cache for resumable runs
- `*.tif` — intermediate and final raster surfaces (~3 GB total)
- `01_*.csv`, `02_*.csv` — large training-table intermediates

To regenerate everything from scratch, run the pipeline in order:

```r
source("R/00_setup.R")
for (s in sprintf("R/%02d_*.R", 1:10)) source(Sys.glob(s)[1])
```
